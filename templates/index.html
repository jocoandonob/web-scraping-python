API Documentation

Endpoints

  • POST /api/scrape
    Scrape a URL
  • GET /api/config
    Get scraper config
  • GET /api/rate-limit
    Check rate limit

Example API Request

POST /api/scrape
{
  "url": "https://example.com",
  "scrape_type": "text",
  "selector": "div.content",
  "config": {
    "timeout": 30,
    "user_agent": "Custom User Agent"
  }
}
